About data storage
This app uses Firebase Firestore (Google's free database) to store all medicine logs in the cloud. The free tier supports up to 50,000 reads and 20,000 writes per day — more than enough for this tracker.

Setup (one time, ~5 min):
  • Go to console.firebase.google.com, create a free project
  • Add a Web App, copy the firebaseConfig object
  • In this HTML file, replace the FIREBASE_CONFIG placeholder near the bottom of the <script> with your config
  • Enable Firestore Database in the Firebase console (start in test mode)
  • Deploy the updated file — all devices will now share data automatically
Until Firebase is configured, the app falls back to localStorage (works on a single browser, no sharing). Once Firebase is set up, all data syncs across devices in real time.
